# Plan de Desarrollo

## Fase 1: MVP (Completada)
- Pipeline de noticias RSS → Supabase
- Validación IA con Gemini
- Frontend React con verificador
- Detección de fake news y contenido patrocinado

## Fase 2: Mobile + Monetización (En curso)
- App Android con Capacitor
- Google AdSense (ads nativos en feed)
- Modelo Freemium (Contexto Claro Pro)

## Fase 3: Escalamiento
- API de verificación pública
- Push notifications
- Newsletter diario
- App iOS
- Más países (AR, PE, MX)

## Fase 4: Producto Comercial
- Venta de API a medios
- Suscripciones enterprise
- White-label para redacciones
